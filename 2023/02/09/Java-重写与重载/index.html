重载：overload

同一个类中，方法名称相同，传入的参数名称不同才是方法重载

❗注意：
（1）在main中第一行new的对象一定要和文件名称一致。
（2）在第一个test()中return了1，因为main中直接用打印的方式获取第一个test()，所以return的内容也会被打印出来。
（3）第二个test()就被重载了，但是注意参数不同，和第一个test()不是一模一样的，需要传入参数的类型、个数、顺序不同
